# 什么是LabVIEW

## LabVIEW与文本编程语言的区别

区别

- LabVIEW是图形编程语言
- 容易学习

相同

- 有着类似其他编程语言的逻辑

## G语言

G语言即图像化编程语言（Graphical Programming Language）

LabVIEW是一种开发环境（类似于Vistual Studio），G语言就是在这种开发环境下编写的代码

## LabVIEW应用领域

- 测试测量
  - 测试仪器
  - 数据采集设备
- 控制领域
  - LabVIEW DSC
- 仿真
  - 模拟，仿真，原型设计
  - 设计机电设备
- 快速开发
- 跨平台
  - Windows
  - Mac OS
  - Linux
  - 各种实时操作系统
    - PDA
    - FPGA
  - 各种嵌入式设备
    - VxWorks
    - PharLap
