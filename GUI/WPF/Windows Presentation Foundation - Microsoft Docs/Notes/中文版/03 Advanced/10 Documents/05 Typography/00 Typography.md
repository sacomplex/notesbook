# [Typography](https://docs.microsoft.com/en-us/dotnet/framework/wpf/advanced/typography)

Windows Presentation Foundation (WPF) 还支持丰富的文本内容演示。 WPF 中的文本通过 Microsoft ClearType技术呈现，该技术增强了文本的清晰度和可读性。 WPF 此外支持OpenType字体，它提供的定义之外的其他功能TrueType®格式。
